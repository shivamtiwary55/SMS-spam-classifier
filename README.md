# SMS Spam Classifier

## What is this model?
This is a Machine Learning model that checks whether an SMS message is **Spam** or **Not Spam (Ham)** using text analysis.

The model is trained on labeled SMS data and learns patterns commonly found in spam messages.

## How to use this project

### Step 1: Install required packages

### Step 2: Run the application

### Step 3: Enter an SMS message
Type or paste any SMS message into the app.

### Step 4: Get prediction
The model will show whether the message is **Spam** or **Not Spam**.

## Files used
- `app.py` – runs the application
- `model.pkl` – trained machine learning model
- `vectorizer.pkl` – converts text into numbers
- `spam.csv` – dataset
